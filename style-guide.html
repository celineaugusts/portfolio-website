Paragraphs

CSS selectors:

p

Sample code:

<p>
 Twice or thrice had I lov'd thee, / Before I knew thy face or name; / So in a voice, so in a shapeless flame / Angels affect us oft, and worshipp'd be; / Still when, to where thou wert, I came, / Some lovely glorious nothing I did see.
</p>
